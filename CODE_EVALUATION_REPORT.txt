1. PROJECT OVERVIEW
================================================================================

The ARBSV website is a multi-page static website for a basketball association
in São Vicente, Cabo Verde. It includes:

- Home page (index.html) - Main landing page
- Campeonato page (basketballpages/Campeonato.html) - Championship tournament
- Abertura page (basketballpages/abertura.html) - Opening tournament

Tech Stack:
- Pure HTML5/CSS3/JavaScript (Vanilla)
- No build tools required
- Uses Google Fonts (Montserrat, Oswald)
- Serves with Node.js (serve package)

================================================================================

--------------------------------------

2.4 DATA MANAGEMENT ⚠️ NEEDS IMPROVEMENT
--------------------------------------
- Data spread across multiple files (games.js, players.js, standings.js)
- Some duplicate data between files
- Inconsistent team naming conventions across files
