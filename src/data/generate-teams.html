📥 Carregar Dados do Excel

Clique no botão abaixo para carregar todos os dados das equipas dos ficheiros CSV e gerar os ficheiros JavaScript.

📁 Estrutura dos Ficheiros CSV

Cada equipa tem o seu próprio ficheiro CSV que pode ser editado no Excel:

Como editar no Excel:

  1. Abra o ficheiro CSV da equipa (ex: src/data/Campeonato/Senior/todos-blacks.csv)
  2. Edite os dados no Excel (nome, número, posição, estatísticas)
  3. Guarde o ficheiro como CSV
  4. Clique em "Gerar Ficheiros JS" para atualizar
💡 Dica: A primeira linha contém os cabeçalhos. A segunda linha é info da equipa. As linhas seguintes são os jogadores.

🔄 Atualizar Página

Após gerar os ficheiros JS, atualize a página do Campeonato para ver as alterações.
